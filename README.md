# vuemvc

Typescript sample using vue CLI, vuex, decorators and dotnet core with entityframework core for backend

## requirements
* dotnet core 2.2
* nodejs v11.3

## build and run with
api: 
* dotnet restore
* dotnet ef database update
* dotnet run

vuets: 
* npm install
* npm run serve

start browser with http://localhost:8080/
