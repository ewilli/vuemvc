# vuemvc

Typescript sample using Vue CLI, Vuex and DotNet Core with Entityframework Code backend

## requirements
* dotnet core 2.2
* notejs v11.3

## build and run with
api: 
* dotnet restore
* dotnet ef database update
* dotnet run
vuets: 
* npm install
* npm run serve

start browser with http://localhost:8080/
